# wormhole

A custom networking layer for Postman.

## C++ library

### Install dependencies

```sh
brew bundle
```

### Configure

```sh
cmake -S . -B build
```

### Build

```sh
cmake --build build
```

### Test

```sh
build/test/request_unit_test
```

## NPM package

### Install dependencies

```sh
npm ci
```
### Build

```sh
npm run build
```

### Test

```sh
npm test
```
